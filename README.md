# SmartNotes - 个人知识管理系统

> 一个为学生、开发者打造的轻量级知识笔记管理平台，支持 Markdown 编辑、标签分类、搜索与未来 AI 辅助。

## 🔧 技术栈
- 前端：Vue 3 + Pinia + Tailwind CSS
- 后端：Flask (Python) / Node.js (Express)
- 数据库：SQLite / MongoDB
- AI 接口（可选）：OpenAI GPT-4o

## ✨ 功能模块
- Markdown 富文本编辑
- 标签分类管理
- 搜索功能
- 数据持久化存储
- GPT 辅助总结（可选）

前端：
```bash
cd frontend
npm install
npm run dev
